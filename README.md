
# PrivIndex AI — Hackathon MVP

منصة تجريبية تنشئ مؤشر تقييم مركب حي لشركات خاصة، ثم تسمح ببناء عقود افتراضية
تُسوّى نقدًا وفق حركة المؤشر أو تقلبه، دون ملكية في الشركات الأساسية.

## ما الذي تعرضه النسخة؟

- بيانات تجريبية لست شركات خاصة.
- مؤشر مركب يبدأ من 100.
- أوزان قابلة للتعديل لمؤشرات الإيرادات، الربحية، السيولة، النمو، الدين، التنفيذ والحوكمة.
- Isolation Forest لرصد القراءات غير المعتادة.
- تفسير مبسط لأكبر مساهم إيجابي وسلبي.
- محاكاة احتمالية لمسار المؤشر.
- مختبر لعقود:
  - Index Future
  - Call Option
  - Put Option
  - Volatility Swap
- تسوية نقدية افتراضية ومنحنى Payoff.

## هيكل العرض في الهاكاثون

1. المشكلة: الشركات الخاصة لا تمتلك سعرًا لحظيًا شفافًا، والمستثمر لا يستطيع تداول تعرضه لسلة منها.
2. الحل: AI يحول البيانات المالية والتشغيلية إلى مؤشر حي قابل للتدقيق.
3. الابتكار: عقود نقدية تتبع المؤشر دون نقل ملكية الشركات.
4. العرض المباشر:
   - غيّر أوزان عوامل التقييم.
   - أدخل أو أخرج شركة من السلة.
   - شاهد المؤشر يعاد حسابه.
   - أنشئ عقدًا وغيّر سيناريو الاستحقاق.
5. الحوكمة: أبرز أن النسخة الحقيقية تحتاج جهة مستقلة لحساب المؤشر وتنظيمًا قانونيًا.
